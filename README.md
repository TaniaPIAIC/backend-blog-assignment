# Backend Blog Assignment

## 1. JWT (JSON Web Token)
JWT is used for login authentication. When a user logs in, the server verifies credentials and provides a token used for future requests.

## 2. Mongoose
Mongoose is a library used to connect Node.js with MongoDB. It helps perform CRUD operations using schemas and models.

## 3. bcrypt
bcrypt is used to hash passwords for security. It prevents storing real passwords in the database.

## 4. Client Side Rendering (CSR)
CSR builds the page in the browser using JavaScript. Example: React apps.

## 5. Server Side Rendering (SSR)
SSR builds the page on the server and sends ready HTML. Example: Next.js.

## 6. MVC Architecture
MVC divides application into Model, View, and Controller for better structure.

## 7. Schema
Schema defines the structure of data in the database.

## 8. Middleware
Middleware runs between request and response to process or check data.
